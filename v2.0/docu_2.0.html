Documentation for version 2.0 (a general model)
1. The MSSM model is as described in micromegas_1.3
  The package also includes routines for g-2, b to s gamma, b_s to mu mu, experimental direct LEP constarints,..
2. The NMSSM model now uses the Susy Les Houches Accord 2 , for a description and physics studies see this paper.
      The NMSSM file uses NMHDECAY and includes LEP constraints.
3. The CPV   model  uses CPsuperH. A   routine for the electron and thallium edm (at two-loop) is included.
4. Implementation of new models first requires to write a new CalcHEP model file.
For more information see the README file

5. If you are using this program please refer to the standard micromegas manual   and the 2.0 update document.

Present version is micromegas 2.0.7 (see list of changes from 2.0)
